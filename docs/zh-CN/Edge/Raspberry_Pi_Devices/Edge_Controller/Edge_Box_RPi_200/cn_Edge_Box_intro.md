EdgeBox-RPI-200 不仅是一款工业级树莓派/ Linux 网关和 IIoT 网关，同时也是一款基于树莓派的多合一边缘计算控制器，能够满足多种工业用途。它设计为高扩展性和坚固的工业硬件，配备丰富的 IO 资源，并支持强大的树莓派工业软件生态系统，是智能自动化和工业物联网（IIoT）解决方案的理想选择。
